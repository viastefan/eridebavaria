# eRide Bavaria

Premium European electric mobility — a digital brand experience, not a webshop.

## Pages

| Route | Experience |
|-------|-----------|
| `/` | Cinematic homepage with 3D hero, scroll storytelling |
| `/shop` | Luxury product grid with animated filters |
| `/product/[slug]` | Apple-style PDP with gallery, configurator, story, specs |
| `/compare` | Side-by-side vehicle comparison (up to 4) |
| `/checkout` | Single-page luxury checkout |
| `/account` | Premium customer dashboard |
| `/support/[slug]` | Per-vehicle support hub |

## Features

- **Navigation** — Floating glass navbar, fullscreen mega menu
- **Search** — Apple Spotlight-style overlay (⌘K)
- **Cart** — Luxury side drawer with recommendations
- **Configurator** — Color, wheels, battery, roof, cargo options
- **Compare** — Interactive spec table with highlights
- **Motion** — GSAP scroll triggers, Framer Motion, Lenis smooth scroll
- **SEO** — Product JSON-LD schema, metadata per page

## Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS 4 · Three.js · GSAP · Framer Motion · Lenis

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Keyboard Shortcuts

- `⌘K` / `Ctrl+K` — Open search
